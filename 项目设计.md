# 科大 Web 俱乐部 网站设计文档

## 项目背景

这个网站是为了创建科大的 Web 俱乐部。该俱乐部旨在聚合科大的小伙伴进行 Web coding，初步想法是每周固定时间定期举行活动，大家两两或多人组队去开发一些项目。

---

## 功能模块

### 1. 注册与登录
- 科大学生注册并登录网站
- 建议通过邮箱后缀 `@mail.ustc.edu.cn` 限制注册，确保成员身份

### 2. 项目池（增强版）
- 成员提交自己的项目想法或需求
- 项目支持打标签（难度、技术栈、所需人数）
- 支持点赞 / 收藏感兴趣的项目
- 活动举行时从项目池中选取项目供大家参与开发

### 3. 活动管理
- 发布每周活动公告（时间、地点、本期项目）
- 成员在线报名
- 历史活动归档与回顾

### 4. 投票系统
- 活动前让成员对项目池中的候选项目进行投票
- 根据投票结果决定本期开发项目

### 5. 成员主页
- 展示个人技术栈、Github 链接
- 记录参与过的项目和活动历史

### 6. 活跃贡献记录
- 追踪成员的参与次数与贡献
- 激励活跃成员（如排行榜、徽章等）
